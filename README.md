# Tema escuro para VS Code

## Instalação

1. Abra **Extensions**: `Ctrl+Shift+X`
2. Busque **andrleo theme**
3. **Install**
4. Ative: `Ctrl+K Ctrl+T` → escolha **andrleo-theme**

| Papel | Cor |
